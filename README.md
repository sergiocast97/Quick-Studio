# Quick Studio

Sergio Castillo's Honours Project
Real-Time, Collaborative Music Recording

## Setup

To set up the app in a local environment:

### Install dependencies:
```
$ npm install
```

### Run the App:
```
$ DEBUG=quick-studio:* npm start
```

### Default URL:
```
http://localhost:3000/
```
